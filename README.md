# Mechrevo-Tongfang-Andio-Senary-Nahimic-Driver-FIX-Sampling-rate
This audio driver fixes the issue where Mechrevo and Tongfang models couldn't be set to 24-bit 192000Hz under Windows 11. It supports Senary audio chips such as the CX11970 and CX11970EVK series. Other series are also supported, including XMG EVO E15 E14, WUJIE 15 series, WUJIE 14 series, JIAOLONG16 series, YAOSHI series, etc.
